RMVL Parameters amd CMake 扩展使用说明

为谁提供服务？

该扩展为 RMVL 以及使用 RMVL 的项目提供服务，有关 RMVL 的介绍可参考下面的内容：

  • RMVL 用户手册
  • RMVL 说明文档
  • Github 地址

该扩展插件支持什么操作？

RMVL 部分 CMake 函数

  • 提供了一组形如 rmvl_* 的函数
  • 代码块：FindRMVL ，输入即可快速包含必要的包
  • 输入：Ctrl + I 或使用 Ctrl + Shift + P 键入 Trigger Suggest 可触发建议

RMVL Parameter 参数规范文件语法

  • 包含基本的类型、变量定义、内置成员函数的语法高亮与补全
  • 代码块：ParaHelp ，输入即可快速浏览示例代码
  • 输入：Ctrl + I 或使用 Ctrl + Shift + P 键入 Trigger Suggest 可触发建议
